#MessageBoard on AppEngine

GoogleAppEngineでメッセージボードを作成するサンプルです。

内輪での勉強会のサンプルとして作成しました。  
AppEngine/Py でDjangoの基本セットアップや、モデル・フォーム利用、  
データベースのつなぎ込みなどがひと通り入ってます。

### 入ってる要素
 * AppEngine/Python
 * Django
   * Model
   * Form
   * Template
 * NDB (BigTable)
 * Googleユーザー認証
